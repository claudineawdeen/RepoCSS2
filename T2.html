overflow : visible

The CSS overflow: visible property is the default value for the overflow property in CSS. It dictates how content that exceeds the boundaries of its containing element should be handled. When overflow: visible is applied (or implicitly used, as it's the default), any content that overflows the element's defined width and height will simply render outside of that element's box. It will not be clipped, hidden, or cause scrollbars to appear. any content that overflows the element's defined width and height will simply render outside of that element's box.
